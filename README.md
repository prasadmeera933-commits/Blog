# 📖 Mindful Mosaic — Modern Blog Website

A modern and interactive **blog website** built using HTML, CSS, and JavaScript.
It includes search, category filters, tags, comments, social sharing, and a text-to-speech feature to listen to articles.

## ✨ Features

* Blog post listing with images
* Category filtering
* Tag filtering system
* Search posts feature
* Full article view
* Text-to-Speech (Listen to article)
* Social sharing (Twitter, Facebook, LinkedIn, WhatsApp)
* Copy link feature
* Comment system (stored in browser localStorage)
* Responsive design
* Clean modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Web Speech API (Text-to-Speech)
* Local Storage (for comments)

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the folder.
3. Open **index.html** in your browser.

## 🔊 Special Feature: Listen to Article

Users can listen to blog posts using the browser's built-in **Text-to-Speech** system.
